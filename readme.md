## web as it should be

`wasm` file generated using `client-app`.  
`brotli` file is created from `wasm` file in `levo-server`.  


To start demo server:

`cd levo-server`  
`cargo r --release`

To run client:

`cd levo-portal`  
`cargo r --release`
