# LEVO: web as it should be

`wasm` file generated using `levo-wasm-generator`.
`brotli` file is created from `wasm` file in `levo-server`

To run it:

`cd levo-server`  
`cargo r --release`

to start demo server.

To run client:

`cd levo-client`  
`cargo r --release`
